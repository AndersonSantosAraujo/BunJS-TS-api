# bunjs-ts-api

Powered By:

- BunJS
- ExpressJS
- Typescript
- PrismaJS

### Prisma Studio

run in http://localhost:5555/

```bash
bun prisma studio
```

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run src/index.ts
```

This project was created using `bun init` in bun v1.0.31. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
